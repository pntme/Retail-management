## Key User Interactions

### 1. Login Flow
```
1. User opens http://localhost:3001
2. Sees beautiful login screen
3. Enters: admin / admin123
4. System validates credentials
5. JWT token generated and stored
6. Redirected to Dashboard
```

### 2. Add Customer Flow
```
1. Click "Customers" in sidebar
2. Click "+ Add Customer" button
3. Modal pops up with form
4. Fill in customer details
5. Click "Create"
6. API creates record in database
7. Modal closes
8. Customer appears in table instantly
```

### 3. Process Sale Flow
```
1. Click "Sales" in sidebar
2. Click "+ New Sale"
3. Large POS modal opens
4. Click products to add to cart
5. Adjust quantities with +/- buttons
6. Optionally select customer
7. Add discount or tax
8. Click "Complete Sale"
9. Inventory automatically updated
10. Sale recorded in database
11. Receipt data available
```

### 4. Recurring Service Flow
```
1. Click "Services" in sidebar
2. Click "+ Add Service"
3. Select customer from dropdown
4. Enter service details
5. Choose billing cycle (weekly/monthly/etc)
6. Set start date
7. Click "Create"
8. Service appears in active services
9. Will generate recurring bills automatically
```
